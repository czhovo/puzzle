图片拼图工具 - 拖放规则

符号说明：
  - 格子坐标 (r, c): r行 c列
  - 对于格子A，其左侧格子为A-1，右侧格子为A+1（同行）
  - 标准图片占1格，宽幅图片占水平连续2格，其起始格记为A，占(A, A+1)

拖拽来源类型：
  unplaced: 从右侧图片列表拖出
  grid: 从左侧网格中拖出

目标区域确定：
  若拖动的图片为标准：目标区域为鼠标释放点所在的单个格子，记作C
  若拖动的图片为宽幅：
     令T为鼠标释放点所在格子
     若T存在且网格列数>1：
        若T不是最右列，则 C=T, D=T+1
        若T是最右列，则 C=T-1, D=T
     否则无法放置(目标无效)

交换定义：
  交换两个区域的内容。区域可以是单格(标准图片)或双格(宽幅图片)。
  其中一个区域可为空，则效果为移动。
  宽幅图片必须整体移动/交换，其grid_start跟随更新。

核心规则：

1. 来自右侧的图片(unplaced)
   1.1 标准图：
       若C为空 → 放入C
       否则 → 禁止
   1.2 宽幅图：
       若C和D均为空 → 放入(C,D)
       否则 → 禁止

2. 来自左侧网格的标准图片(grid, 原在格子A)
   2.1 C为空 或 C内为标准图片 → 交换A与C
   2.2 C为宽幅图的左半部分(即宽幅占据(C, C+1)):
       若A+1存在且为空 → 交换区域(A, A+1)与(C, C+1)
       否则若A与宽幅图左侧相邻(A = C-1) → 强制交换区域(A, A+1)与(C, C+1)   // 特殊情况:相邻标准-宽幅直接交换
       否则 → 禁止
   2.3 C为宽幅图的右半部分(即宽幅占据(C-1, C)):
       若A-1存在且为空 → 交换区域(A-1, A)与(C-1, C)
       否则若A与宽幅图右侧相邻(A = C+1) → 强制交换区域(A-1, A)与(C-1, C)   // 特殊情况
       否则 → 禁止

3. 来自左侧网格的宽幅图片(grid, 原占据(A, B) 其中B=A+1)
   (目标为(C,D)，由鼠标位置按前述宽幅目标算法得到)
   3.1 (C,D)均不被宽幅图占用(即均为空或为标准图) → 交换区域(A,B)与(C,D)
   3.2 (C,D)恰好是一张完整的宽幅图(C为其起始格) → 交换两宽幅区域
   3.3 否则(至少一格被其他宽幅图部分占用且不构成同一张完整宽幅) → 禁止

4. 拖回右侧
   来自左侧的任何图片，若释放时鼠标在右侧画布区域内：
       标准图：从A移至右侧对应文件夹
       宽幅图：从(A,B)移至右侧，两格清空
   来自右侧的图片放回右侧画布：视为取消，无变化。

5. 无效区域
   释放时鼠标既不在左侧格子也不在右侧画布：拖拽取消，图片归位。
